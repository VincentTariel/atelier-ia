# Checklist “démarrage express” — Scénario A (ateliers privés / prestation) — Nouvelle-Calédonie

Objectif : **être prêt à encaisser ta première vente** avec le minimum d’administratif, puis rester clean (factures, patente, suivi).

---

## 0) Tant que tu ne factures pas (phase test)
- [ ] Pub Facebook / site / page : OK
- [ ] Pré-inscriptions **sans paiement** : OK
- [ ] Message clair : “pré-inscription sans engagement, pas de paiement”
- [ ] Tu évites : encaisser un acompte / vendre une date / émettre une facture **avant** d’avoir ton EI + RIDET

---

## 1) Le déclencheur : “j’ai un client prêt à payer”
Dès que tu as :
- un devis accepté **ou**
- un acompte demandé **ou**
- une date d’atelier vendue

➡️ Tu passes en **Entreprise Individuelle (EI)** + **RIDET** + **patente**.

---

## 2) Infos à préparer (10 minutes, tout en un)
### Identité / contact
- [ ] Nom + prénom
- [ ] Date et lieu de naissance (si demandé)
- [ ] Téléphone, email pro
- [ ] Adresse de domiciliation (siège) : **ton adresse** ou une adresse pro

### Activité (libellé simple)
- [ ] Activité principale (exemples) :
  - “**Ateliers pratiques d’outillage numérique / IA** pour entreprises”
  - “**Conseil et accompagnement à la mise en place d’outils IA**”
  - “**Prestations de formation privée** (hors dispositif de formation pro continue)”
- [ ] Lieu d’exercice : Nouméa (ou “chez le client” si prévu)

### Date de début d’activité
- [ ] Choisis une **date proche de la 1ère session payante** (idéalement début de mois pour simplifier)

### Banque / paiement
- [ ] Compte bancaire (perso au départ, puis pro si besoin)
- [ ] Moyen de paiement : virement / espèces / carte (si tu as un TPE)

---

## 3) Création EI (le plus simple)
- [ ] Créer l’**Entreprise Individuelle** via le **Guichet Entreprises NC**
- [ ] Récupérer ton **numéro RIDET** (à mettre sur devis/factures)

> Astuce : fais la création **juste avant** ta première facturation, pas 2 mois trop tôt.

---

## 4) Patente (ne pas oublier)
- [ ] Déclarer la **patente** dans le délai légal après le début d’activité
- [ ] Noter dans ton calendrier : *patente + échéances fiscales* (si tu continues l’activité)

> Le point important : une fois en activité, si tu arrêtes, il faut penser à la **radiation** (RIDET + fiscal) pour ne pas rester “actif” administrativement.

---

## 5) Kit administratif prêt à copier-coller

### A) Mentions à mettre sur tes documents (minimum)
Sur **devis** et **factures** :
- [ ] Nom / Prénom ou Nom commercial
- [ ] Adresse
- [ ] Téléphone + email
- [ ] **RIDET : XXXXXXX.XXX**
- [ ] Date + numéro de document
- [ ] Désignation de la prestation (ex: “Atelier IA – 3h – paramétrage assistant + modèles”)
- [ ] Prix (XPF) + conditions de paiement
- [ ] (Option) Mention TGC : **à vérifier selon ton régime**  
  Exemple courant si franchise : “TGC non applicable (franchise en base)”.

### B) Numérotation simple
- [ ] Devis : `D-2026-001`, `D-2026-002`…
- [ ] Factures : `F-2026-001`, `F-2026-002`…

### C) Archive (anti-galère)
- [ ] Un dossier “Admin” (Drive/Dropbox) :
  - `01_Devis/`
  - `02_Factures/`
  - `03_Paiements/`
  - `04_Documents_officiels/` (RIDET, patente, etc.)
- [ ] Conserver : devis signé + facture + preuve paiement

---

## 6) Modèles (à utiliser tels quels)

### Modèle 1 — Texte “pré-inscription”
> Pré-inscription **sans engagement**.  
> Aucun paiement n’est demandé à ce stade.  
> Les personnes pré-inscrites seront contactées en priorité dès l’ouverture des premières sessions.

### Modèle 2 — Devis (structure)
**DEVIS N° D-2026-001**  
Date : JJ/MM/AAAA  
Client : [Nom entreprise] — [Nom contact] — [Adresse]  

**Objet : Atelier IA (présentiel) – mise en place d’assistants IA sur-mesure**  
- Durée : X heures  
- Contenu : diagnostic tâches, configuration assistant, templates, prise en main  
- Livrables : prompts + checklists + structure d’assistant  
- Lieu : [chez le client / salle / Nouméa]  

**Prix : XX 000 XPF**  
Modalités : [paiement à réception / acompte X% / virement]  
Validité du devis : X jours  

Émetteur : Vincent Tariel — RIDET : XXXXXXX.XXX — [coordonnées]

Signature client (bon pour accord) : ………

### Modèle 3 — Facture (structure)
**FACTURE N° F-2026-001**  
Date : JJ/MM/AAAA  
Client : [Nom entreprise] — [Adresse]  

Prestation : Atelier IA – [détails]  
Montant : XX 000 XPF  
Paiement : [virement / espèces]  
Échéance : [date]  

Émetteur : Vincent Tariel — RIDET : XXXXXXX.XXX — [coordonnées]  
Mention TGC : [selon régime]

---

## 7) Mini-check “à faire avant la 1ère session”
- [ ] EI créée
- [ ] RIDET reçu
- [ ] Devis modèle prêt + facture modèle prêt
- [ ] Tarif final défini
- [ ] Conditions simples écrites (annulation, report, paiement)
- [ ] Patente notée dans ta to-do (avec rappel calendrier)

---

## 8) Quand tu grandis (pas maintenant)
- Compte bancaire pro
- CGV plus solides
- Déclaration “prestataire de formation pro continue” uniquement si tu bascules vers le financement / conventions FPC

---

### Notes rapides (pour rester simple)
- **Scénario A** : tu vends des ateliers privés / prestations — tu évites le formalisme “organisme de formation” tant que tu n’en as pas besoin.
- Si un client te demande “formation prise en charge / financement” : ce sera souvent le signal pour envisager le cadre FPC (plus tard).
